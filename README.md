# PromptVerse

PromptVerse is a full-stack web application that allows users to explore and manage trending AI prompts for tools like ChatGPT and Google Gemini.

## 🚀 Features

- Upload prompts with images
- Categorize prompts (ChatGPT, Gemini, etc.)
- Admin dashboard for managing content
- Image upload using Cloudinary
- JWT-based authentication
- Protected admin routes using middleware
- Responsive UI with modern design

## 🛠 Tech Stack

- Next.js (App Router)
- MongoDB + Mongoose
- Tailwind CSS
- Cloudinary (image storage)
- JWT Authentication

## 📂 Core Functionality

- Create, read, and delete prompts
- Link prompts with categories (foreign key relationship)
- Populate category data using MongoDB
- Secure admin access with middleware

## 📌 Use Case

This platform helps users find high-quality, trending prompts to generate better AI outputs, especially for image generation and creative content.
